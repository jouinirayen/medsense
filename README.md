MedSense – Integrated Healthcare Platform
Project Description

MedSense is a comprehensive web platform designed to unify and simplify healthcare management. Developed as part of an academic project at ESPRIT, it addresses the fragmentation of medical services by providing a holistic ecosystem for patients, doctors, and administrators.

Core Features:

Centralized management of medical services

Appointment scheduling and tracking

Patient and doctor dashboards

Health record management

Notifications and alerts

Role-based access (Admin, Doctor, Patient)

Technology Stack:

Backend: PHP (MVC architecture)

Database: MySQL (using PDO)

Frontend: HTML, CSS, SCSS, JavaScript

Server Environment: XAMPP (Apache + MySQL)

Installation

Clone the repository

git clone https://github.com/Eyatallah-14/medsense.git


Navigate to the project folder

cd medsense


Setup a local server

Install XAMPP (or equivalent)

Start Apache and MySQL

Move the project to the server directory

Place the project inside htdocs (for XAMPP)

Database setup

Open phpMyAdmin

Create a database named medsense

Import the provided SQL file

Configure database connection in the project (check config/database.php or similar)

Usage

Access the Application:

Main page: http://localhost/medsense/

Login page: http://localhost/medsense/view/frontoffice/login.php

Main Functionalities:

User Registration & Login – secure authentication for patients and doctors.

Role-based Access:

Admin

Doctor

Patient

Appointment Management – schedule, view, and manage appointments.

Patient Records – maintain and access medical records securely.

Notifications & Alerts – email notifications for appointments and updates.
